# Django API

Very basic API using Django and Django Rest Framework.

# Instruction
To run this, install all the requirements (OPTIONAL: use virtual env)  
`pip install -r requirements.txt`  

Run migrations  
```
$ ./manage.py makemigrations
$ ./manage.py migrate
```

Run server  
`./manage.py runserver`
